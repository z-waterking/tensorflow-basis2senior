# tensorflow-basis2senior

## 说明

本代码库所用的python版本为Anaconda包中的2.7

tensorflow环境为：1.10

本教程假设你已经了解了关于**python**和**深度学习**的基本知识，例如numpy的使用，矩阵运算、向量，多层感知机等

## 什么是好代码？

本库的目标在于写"笨代码"。

其间有两个要素，清晰的代码结构与完善的代码文档。

写最显式的代码，~~吹最狠的NB~~

## 深度学习一般套路

1. 加载数据
2. 构造输入
3. 搭建网络
4. 计算输出
5. 计算loss
6. 反向传播
7. 打完收工
